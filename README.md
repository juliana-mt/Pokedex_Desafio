Desafio Pokédex

1. Descrição
Uma aplicação criada como desafio para consumir a PokéAPI e construir uma Pokédex completa, permitindo visualizar informações dos Pokémons e gerenciar uma lista de favoritos. A aplicação salva os Pokémons favoritados diretamente no LocalStorage, garantindo que permaneçam mesmo após recarregar a página. 

2. Tecnologias utilizadas
    - Vue.js 3
    - Vite
    - TailwindCSS
    - Axios / Fetch API
    - PokéAPI

3. Instruções de instalação e execução
   1. Clone o repositório: https://github.com/juliana-mt/Pokedex_Desafio
   2. Acesse a pasta do projeto
   3. Instale as dependências no terminal: npm install
   4. Inicie o servidor no terminal: npm run dev
   5. Abra o navegador e acesse: http://localhost:5173
